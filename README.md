# Tellagen Plugins for Claude Code

Official plugin marketplace for [Tellagen](https://tellagen.com).

## Install

```bash
/plugin marketplace add tellagen/tellagen-plugins
/plugin install tellagen@tellagen-plugins
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| **tellagen** | Incident investigation — MCP server + `/tellagen:investigate` skill |

## What you get

- **Tellagen MCP server** — 12 tools for reading incidents, running investigations, posting findings, and promoting to timelines
- **Investigation skill** (`/tellagen:investigate`) — structured workflow that walks Claude through a full incident investigation

## Prerequisites

- [Claude Code](https://claude.ai/claude-code) v1.0.33+
- A Tellagen workspace with an API key ([Settings > API Keys](https://app.tellagen.com/settings/api-keys))
